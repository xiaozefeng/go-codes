## struct
- 不论是地址还是结构体本身， 一律使用 . 访问成员
- struct 没有构造方法，可以使用构造函数
- 只有指针才可以改变结构体内容
- nil指针也可以调用方法

## 值接受者 vs 指针接受者
- 要改变内容必须使用指针接收者
- 结构过大也考虑使用指针接受者
- 一致性: 如有指针接受者，最好都是指针接收者

## 封装
- 变量名称一般使用CamelCase
- 首字母大写: public
- 首字母小写: private

## package
- 每个目录一个包
- main包包含了可执行入口
- 为结构体定义的方法必须放在同一个包内, 但是可以是不同的文件
